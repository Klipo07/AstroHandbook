# AstroHandbook — астрономический справочник с 3D-моделью Солнечной системы

Android-приложение — интерактивный астрономический справочник с 3D-визуализацией Солнечной системы на OpenGL ES. Пользователь может перемещаться между планетами, просматривать информацию о каждой из них и наблюдать за орбитальным движением в реальном времени.

## Возможности

- **3D-модель Солнечной системы** — Солнце и планеты с орбитальным вращением, Луна вращается перпендикулярно плоскости эклиптики
- **Навигация по планетам** — кнопки «влево» и «вправо» перемещают выделение между планетами и спутниками; выбранный объект подсвечивается полупрозрачным кубом
- **Справочные карточки** — описание и изображение для каждой планеты, открываются по нажатию кнопки «информация»
- **Освещение по модели Фонга** — при выборе Луны отображается детализированная модель с реалистичным освещением
- **Текстурированный фон** — квадрат с текстурой звёздного неба, отмасштабированный как фоновое изображение
- **Водная поверхность** — поверхность Нептуна заменена на анимированную водную гладь с волнами
- **Анимация чёрной дыры** — пролетает на фоне галактики на главном экране
- **Лента новостей** — экран с 4 новостями, автоматическая ротация каждые 5 секунд, система лайков
- **Jetpack Compose + ViewModel** — современный UI-фреймворк для экранов приложения

## Стек технологий

| Технология | Применение |
|---|---|
| **Kotlin** | Основной язык разработки |
| **OpenGL ES** | 3D-рендеринг: планеты, орбиты, текстуры, освещение по Фонгу |
| **Jetpack Compose** | UI-компоненты: экран новостей, навигация |
| **ViewModel** | Управление состоянием, сохранение данных при повороте экрана |
| **Android SDK** | Платформенные API |
| **Gradle (Kotlin DSL)** | Система сборки |

## Запуск

**Требования:** Android Studio, JDK 17+, Android SDK

```bash
git clone https://github.com/Klipo07/AstroHandbook.git
```

Откройте проект в Android Studio: **File → Open → выбрать папку AstroHandbook**.

Или соберите из терминала:

```bash
cd AstroHandbook
./gradlew assembleDebug
```

APK будет в `app/build/outputs/apk/debug/`.

## Ключевые технические решения

- **OpenGL ES rendering pipeline** — кастомные шейдеры для модели освещения Фонга (ambient + diffuse + specular)
- **Процедурная анимация** — орбитальное вращение планет, волновая анимация водной поверхности Нептуна
- **Граф сцены** — иерархия объектов: Солнце → планеты → спутники с независимыми матрицами трансформации
- **Текстурирование** — наложение текстур на 3D-примитивы (сферы, квадраты), управление UV-координатами
- **Jetpack Compose + OpenGL** — интеграция декларативного UI с низкоуровневой 3D-графикой в одном приложении
